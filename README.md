# Complete Playwright Framework (POM + Cucumber + Allure + Waits + World + Teardown)

## Install
```
npm install
```

## Run tests
```
npm test
```

## Generate Allure report
```
npm run allure:report
npm run allure:open
```

## Features
- Page Object Model
- Cucumber BDD
- Allure reporting
- Global waits after each step
- Clean teardown (browser closes correctly)
- Custom Cucumber World
- Fully working on Windows & Mac
